### NaN값 채우는 방법 
* method = 'ffill' or 'pad' 
 ```python
df3 = df.reindex(['a','b','c','d'], method = 'ffill')

df4 = df.reindex(['a','b','c','d'], method = 'pad')
```
* method = 'bfill' or 'backfill' 

```python
df3 = df.reindex(['a','b','c','d'], method = 'bfill')

df4 = df.reindex(['a','b','c','d'], method = 'backfill')
```
